6. EXIGENCES TECHNIQUES (SEO + PERFORMANCE)
6.1. SEO technique
* Balises <title> uniques
* Balises <meta name="description"> optimisées
* Un seul <h1> par page
* Structure H2 > H3 cohérente
* Images en WebP compressées
* Texte ALT descriptif obligatoire
* URLs propres, courtes et sans majuscules
* Temps de chargement < 2 secondes
* Sitemap XML + robots.txt
6.2. Performance
* Lazy loading images
* Mise en cache
* Minification CSS/JS
* Hébergement optimisé
* Compatibilité mobile irréprochable
6.3. Accessibilité
* Contraste respecté
* Navigation clavier
* Attributs aria
6.4. CMS
Le site peut être réalisé :
* WordPress (pages statiques + blog)
* Webflow
* Symfony/Twig (sur mesure)
* Autre ?
